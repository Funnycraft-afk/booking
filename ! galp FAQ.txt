node.js - nodejs.org
установка NodeJS и NPM

git console - git-scm.com (git система, в комплекте идёт консоль git bash консоль)

Проверка установленных компонент:

nodejs: node -v

пакетный менеджер nodejs: npm -v

***********************************
(gulpjs.org)

Установка глобальной версии gulp = gulp-cli -g: работает со всеми версиями gulp 3 и 4
(-g - значит глобально в систему)

npm i -g gulp-cli

gulp -v - проверка что gulp установлен глобально

************************************
если запустить консоль в выбранной папке (название папки одним словом или
через дефисы, пробелов лучше избегать)

и ввести npm init - то мы создадим новый проект (новый файл package.json) и пройдя по шагам
введём некоторые параметры (имя проекта, версия, описание, автор и т.д.)

************************************
npm i: установка всех зависимостей из файла package.json

npm i gulp --save-dev - gulp у нас установлен глобально, на компе, а это как бы создание отдельно взятого файла gulp в проекте
создастся папка node_modules
(--save-dev нужно для совместимости проектов на разных ОС или версиях ОС)

************************************

npm i gulp-sass --save-dev - установка плагина gulp-sass

npm i browser-sync --save-dev - установка плагина для обновления браузера

npm i gulp-uglify --save-dev - для минификации

npm i gulp-concat --save-dev - для конкатенации

!!! Важно - normalize.css - лучше ставить отдельно и с ключом --save-dev
npm i normalize.css

slick-carousel
magnific-popup (через пробел/ без --save-dev в конце, т.к это не плагины)

(установка 3-х библиотек, они все есть в npm, их папки появятся в node-modules, файлы css нужно будет подключить из них,
и файлы js)

npm i gulp-rename --save-dev - плагин для переименования

npm i del --save-dev - для удаления файлов и папок

npm i gulp-autoprefixer --save-dev

npm i gulp-file-include --save-dev - вложения в файл из других файлов (@include)

************************************

gulp - запуск gulp

gulp css - запуск отдельной таски, в данном случае запускается таск под названием 'css'

************************************

Дополнительно можно подключить:

npm i animate.css --save-dev - для анимации (прописать в таск 'css' путь для неё)

npm i wow.js --save-dev - для взаимодействия с animate.css

************************************

Библиотеки на чистом JS:

npm i imask --save-dev - маска набора номера телефона

npm i choices.js --save-dev - A vanilla JS customisable select box/text input plugin

npm i flatpickr --save-dev - Стилизация выбора календаря/времени

npm i swiper --save-dev - swiper slider

npm i glightbox --save-dev - Галерея

npm i smooth-scroll --save-dev - Связывание ссылок с разделами на странице (архивный репо, но работает)
